## 项目概述

一个简单易用的MC皮肤编辑器

- 导入皮肤
- 皮肤涂鸦
- 下载皮肤
- 保存草稿
- 显示模型
- 改变人物姿势
- 改变人物模型
- 缩放、移动画布

![Alt text](/public/show.png)

## 运行项目

安装依赖

```
pnpm install
```

运行项目

```
pnpm run dev

```
